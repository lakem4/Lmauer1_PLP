# Personal Landing Page

## Description
This project is a personal landing page built to showcase my projects, technical skills, and experience to recruiters and hiring managers. It provides a centralized place to highlight my work, demonstrate my abilities, and make it easy to connect with me. The goal is to go beyond a traditional resume by showing both my technical skillset and my ability to communicate and collaborate effectively.

## Features
- One-page professional portfolio layout
- Dedicated sections for Bio, Skills, Projects, and Contact
- Highlights real-world projects including dashboards, machine learning, and data analysis
- Clean and simple design focused on readability
- Direct links to GitHub, LinkedIn, and email
- Organized folder structure for scalability and clarity

## Tech Stack

| Technology | Purpose |
|-----------|--------|
| HTML5 | Structure and content of the site |
| CSS3 | Styling and layout |
| JavaScript | Basic interactivity and enhancements |
